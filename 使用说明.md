# 胡哥说餐饮公众号文章生成器使用说明

这套工具用于在本地电脑上生成「胡哥说餐饮」公众号文章、封面图、正文配图和公众号排版，并可以通过微信公众平台接口上传到公众号草稿箱。

## 1. 工具能做什么

- 自动抓取昨天的餐饮相关热点，来源包括 Bing 新闻和抖音热点。
- 根据热点关键词生成公众号文章。
- 标题末尾自动使用连续自然数编号。
- 自动生成公众号封面图。
- 自动生成正文配图。
- 封面图和正文配图会结合文章标题、关键词、摘要和建议内容生成。
- 生成后会检查图片是否正常加载到正文中，并阻止未经允许的外部图片链接。
- 自动生成公众号 HTML 排版。
- 在浏览器后台预览文章、封面图、正文配图和排版。
- 点击按钮上传单篇或一批文章到公众号草稿箱。
- 在浏览器里调整文章角度、关键词、标题风格和配图设定。
- 对生成的本地草稿进行单篇归档、单篇删除、全部归档、全部删除，也可以勾选多篇后批量处理。

## 2. 重要说明

这套工具是本地运行工具，GitHub 只用来保存代码。

公众号的 AppID、AppSecret、OpenAI API Key、生成记录、草稿记录都保存在本机，不会上传到 GitHub。

目前图片生成支持两种方式：

- 本地漫画图：不需要 OpenAI Key，稳定可用。
- OpenAI API 图片：需要 `OPENAI_API_KEY`。

不能直接通过“登录 ChatGPT 网页账号”来稳定自动生成图片。ChatGPT 网页账号和 OpenAI API 是两套体系；如果没有 API Key，建议先使用本地漫画图。

## 3. 启动网页后台

打开 PowerShell，执行：

```powershell

$env:PYTHONPATH="src"
.\.venv\Scripts\python.exe -m huge_catering.webapp --host 127.0.0.1 --port 8765

```

然后在浏览器打开：

```text
http://127.0.0.1:8765
```

如果你用的是 8767，就打开：

```text
http://127.0.0.1:8767
```

## 4. 第一次配置公众号信息

项目根目录有 `.env` 文件，里面需要配置公众号后台信息：

```dotenv
WECHAT_APP_ID=你的公众号AppID
WECHAT_APP_SECRET=你的公众号AppSecret
BRAND_NAME=胡哥说餐饮
COLUMN_NAME=餐饮要赚钱 听我10句劝
AUTHOR_NAME=胡哥
OUTPUT_DIR=outputs
TOPIC_LIBRARY=data/topic_library.json
START_ISSUE_NUMBER=229
ENABLE_TREND_CONTENT=true
```

注意：

- AppSecret 如果看不到，只能在公众号后台重置。
- 微信公众平台后台需要把当前电脑的公网 IP 加入开发者配置的 IP 白名单。
- `.env` 不会上传到 GitHub。

## 5. 打开生成设置

网页后台顶部有两个入口：

- 草稿列表
- 生成设置

点击「生成设置」可以调整这些内容：

### 文章设定

- 标题风格：
  - 热点提醒型
  - 10句劝型
  - 经营信号型
  - 问题钩子型
- 关键词覆盖：
  - 可以手动填写关键词，例如：`外卖,复购,食品安全`
  - 如果不填写，系统会自动抓取昨天热点并总结关键词。
- 文章生成机制/角度：
  - 用来告诉系统文章重点从哪个角度拆解。
  - 例如：`把热点拆成餐饮老板今天能执行的门店动作，重点讲成本、复购和风险控制。`

### 配图设定

- 图片生成方式：
  - 本地漫画图
  - OpenAI 图片生成
- OpenAI API Key：
  - 只有选择 OpenAI 图片生成时才需要填写。
  - 没有 Key 就使用本地漫画图。
- 胡哥形象：
  - 用来定义图片里的固定人物形象。
- 图片风格：
  - 用来定义漫画、插画、颜色、画面元素等。

设置保存后，从下一次生成文章开始生效。

## 6. 生成 10 篇文章预览

回到「草稿列表」页面：

1. 选择起始日期。
2. 填写生成篇数，最多 10 篇。
3. 点击「生成10篇预览」。

生成完成后，页面会显示每篇文章卡片，包括：

- 标题
- 日期
- 选题
- 关键词
- 是否已经上传公众号草稿箱

点击「预览」可以查看：

- 封面图
- 正文配图
- 正文排版

## 7. 上传到公众号草稿箱

预览确认没问题后，有两种上传方式：

- 单篇上传：在某篇文章卡片上点击「上传」。
- 批量上传：点击「上传本批次到公众号草稿箱」。

上传成功后，页面会显示草稿 `media_id`。

上传的是公众号后台草稿，不是直接群发。建议你最后进入公众号后台再人工检查一遍。

## 8. 草稿归档和删除

草稿列表支持这些本地管理功能：

- 单篇归档：把某一篇从当前草稿列表移到归档列表。
- 单篇删除：删除某一篇本地生成文件。
- 勾选多篇后归档：批量移动到归档列表。
- 勾选多篇后删除：批量删除本地生成文件。
- 全部归档：把当前列表里的草稿全部归档。
- 全部删除：把当前列表里的草稿全部删除。

归档文件会移动到：

```text
outputs/_archive/YYYY-MM-DD/
```

注意：这些按钮只管理本地生成文件。如果文章已经上传到微信公众号草稿箱，本地删除不会删除公众号后台里的远程草稿，需要你进入公众号后台手动删除。

## 10. 生成文件在哪里

每篇文章的生成结果保存在：

```text
outputs/YYYY-MM-DD/
```

里面包含：

```text
article.html      公众号正文排版
cover.jpg         封面图
inline-card.jpg   正文配图
metadata.json     文章标题、关键词、编号、草稿ID等信息
```

`metadata.json` 里还有 `image_check` 字段，用来记录正文图片检查结果。正常情况下：

```text
ok=true
local_images=本地原创生成图片
remote_images=上传后只允许微信素材域名
```

## 11. 编号规则

标题末尾使用自然数编号。

起始编号在 `.env` 里配置：

```dotenv
START_ISSUE_NUMBER=229
```

生成记录保存在：

```text
data/publish_history.json
```

同一天重复生成时，会尽量沿用当天编号，避免编号乱跳。

## 12. 避免内容重复

系统会记录已经生成过的选题和标题：

```text
data/publish_history.json
```

热点生成启用时，系统会按昨天热点重新生成文章。

本地内容库模式下，10篇文章用完后，会自动生成新一组内容，并避开历史记录里已经用过的选题。

## 14. 常见问题

### 网页打不开

确认 PowerShell 里的服务还在运行。

如果 8765 被占用，换端口：

```powershell
$env:PYTHONPATH="src"
.\.venv\Scripts\python.exe -m huge_catering.webapp --host 127.0.0.1 --port 8767
```

然后打开：

```text
http://127.0.0.1:8767
```

### 不能上传公众号草稿

检查：

- `.env` 里是否有 `WECHAT_APP_ID`
- `.env` 里是否有 `WECHAT_APP_SECRET`
- 公众号后台 IP 白名单是否加了当前电脑公网 IP
- VPN 是否导致出口 IP 变化

### OpenAI 图片不能生成

检查：

- 是否填写了 `OPENAI_API_KEY`
- 是否在「生成设置」里选择了 OpenAI 图片生成
- API Key 是否有可用额度

如果没有 API Key，使用本地漫画图即可。

### 生成内容重复

可以先删除或备份不想保留的测试输出，但不要随意删除 `data/publish_history.json`，否则系统会忘记历史编号和历史选题。

### 想重新生成某一天

可以在网页里选择同一天重新生成，也可以删除对应目录后再生成：

```text
outputs/YYYY-MM-DD/
```

如果当天已经写入历史记录，编号通常会沿用原来的编号。

## 15. 推荐日常流程

1. 打开 PowerShell 启动网页后台。
2. 浏览器进入草稿列表。
3. 进入生成设置，确认文章角度和配图方式。
4. 回到草稿列表，生成 10 篇预览。
5. 逐篇预览，确认标题、正文、配图。
6. 点击上传单篇或批量上传。
7. 进入公众号后台草稿箱做最终检查。
8. 手动发布或定时发布。
