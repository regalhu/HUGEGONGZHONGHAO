
这个仓库只保存代码和说明文档，不保存你的公众号密钥、OpenAI Key、生成记录和草稿文件。

## 工具主要能力

- 抓取公开可获取的餐饮热点信息。
- 统计引用频次排名前 10 的餐饮相关热点词。
- 分别围绕 10 个热点词生成 10 篇公众号草稿。
- 自动生成封面图、正文配图和公众号排版。
- 支持预览、上传公众号草稿箱、归档、删除、批量处理。
- 文章结果页提供图片生成工作台，可复制提示词到 ChatGPT Plus 手动生成图片。

## 转发给朋友时可以附上的说明

这是一套本地运行的公众号文章生成工具。代码在 GitHub 上，真正运行需要在自己的电脑上安装 Python、配置公众号 AppID/AppSecret，并把自己的公网 IP 加入微信公众号后台 IP 白名单。

朋友使用时需要自己准备：

- Python 3.10 或以上版本。
- 微信公众号 AppID。
- 微信公众号 AppSecret。
- 公众号后台 IP 白名单配置。
- 可选：ChatGPT Plus，用于手动生成图片。

不要把你的 `.env` 文件、公众号 AppSecret、OpenAI API Key、`token_cache.json` 发给别人。

## 朋友本地安装参考

```powershell
git clone https://github.com/regalhu/HUGEGONGZHONGHAO.git
cd HUGEGONGZHONGHAO
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
Copy-Item .env.example .env
```

然后编辑 `.env`，填入自己的公众号信息：

```dotenv
WECHAT_APP_ID=自己的公众号AppID
WECHAT_APP_SECRET=自己的公众号AppSecret
BRAND_NAME=自己的品牌名
AUTHOR_NAME=作者名
START_ISSUE_NUMBER=1
ENABLE_TREND_CONTENT=true
```

启动网页后台：

```powershell
$env:PYTHONPATH="src"
.\.venv\Scripts\python.exe -m huge_catering.webapp --host 127.0.0.1 --port 8766
```

浏览器打开：

```text
http://127.0.0.1:8766
```

## 后续更新代码

在项目目录执行：

```powershell

```

如果本机 GitHub 网络需要代理，本项目已经配置过本地代理：

```powershell
& "C:\Program Files\Git\cmd\git.exe" config --local http.proxy http://127.0.0.1:10809
& "C:\Program Files\Git\cmd\git.exe" config --local https.proxy http://127.0.0.1:10809
```

```powershell
& "C:\Program Files\Git\cmd\git.exe" config --local --unset http.proxy
& "C:\Program Files\Git\cmd\git.exe" config --local --unset https.proxy
```
